Joining variable cases.

Using the "var foo = "Hello World" + "World Hello";" is so 2017! The concat() method is the new way to go!

Concat() has been around for a while but people still don't use it. The concat() method joins 2 or more strings to a pre-existing one. Eg: foo.concat("Hello", foo, 5);. Changing cases is also easier because of the .case functions. Eg: foo.toUpperCase(); HELLO WORLD

Length & Replace.

This is how to find string lengths and replace them:

Length

.length will give you the length of the variable before .length() function is called. Note: The .length() functions will only give you the word count. Eg: foo.length; 2

Replace

.replace("", "") will replace the value of the first double quote to the second double quote. Eg: foo.replace("World", "Hello"); Hello Hello

Quotes

Always use quotes when defining strings. Though "100" may be accepted as a string, Hello World(without the quotes) is, however, not.

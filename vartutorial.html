Declaring Variables.

Variables are represented by letters or special characters that can have their values changed later in code.

Declaring variables is easy. All you need to do is type "var" followed by what you want to name the variable. I'll pick "foo". Then type " = " and then a string or number. Close it with ";". You should have something like this: "var foo = "Hello World";". To change the value of a variable, type the variables name("foo" in my case) followed by " = " and the variables new value. Eg: foo = "World Hello"; . You can use our Math tutorials or String Tutorials for more information.

Console logs.

This is how to get logs from the console:

Open Console

Open the console as you would do normaly by going to Dev Tools> Console. If you are still not sure how to do so, go here.

Create a variable

Create a variable and give it a value. If you are not sure how, go here. If you want to learn about the variables types, go here.

Run logs

Type "console.log();" with your variable's name inside the brackets. "console.info()" is used to achieve the same result.
